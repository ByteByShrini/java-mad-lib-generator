# 🌈 Mad Lib Adventure

A fun and interactive Mad Libs game built using HTML, CSS, and JavaScript.

## 🎮 About the Project

Mad Lib Adventure is a word game where users enter different types of words (nouns, verbs, adjectives, etc.) and generate a funny story based on their inputs.

This project was created to practice:
- HTML structure
- CSS styling
- JavaScript DOM manipulation
- User input handling

## ✨ Features

- Colorful and responsive design
- Interactive user inputs
- Funny story generation
- Beginner-friendly code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
